# Proposal

## Problem

The project needs a small example that shows how Spec to Ship artifacts look.

## Goal

Demonstrate the artifact shape for a simple change.

## Non-goals

This example does not implement real application code.

## Scope

Documentation-only example.

## Risks and Unknowns

None.

## Acceptance Scenarios

- Given a new contributor reads this example
- When they inspect the files
- Then they understand the expected artifact structure
